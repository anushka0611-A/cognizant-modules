Exercise 19: Sass Customization

To customize Bootstrap with Sass:

  1. Run: npm install bootstrap
  2. Create custom.scss and add:
    $primary: #ff6600;
    $border-radius: 1rem;
    @import "node_modules/bootstrap/scss/bootstrap";
  3. Compile with: npx sass custom.scss custom.css
  4. Link custom.css instead of Bootstrap CDN.
